# Network Anomaly Detection For Insider Attacks - Frontend Dashboard
### HIMIT Graduation Project | UI/UX Module

---

## Description
This is the **React.js** implementation of the system's dashboard. It connects to the Flask Backend to visualize real-time network traffic analysis and display security alerts for insider threats.

## Key Features in this Module
* **Real-time Analytics:** Uses `Recharts` for dynamic data visualization.
* **Security Monitoring:** Displays ensemble model metrics (99.92% accuracy).
* **Responsive UI:** Built with `Tailwind CSS` for high performance and sleek dark-mode design.
* **Alert System:** Specialized visual cues for Critical, High, and Medium threats.

## How to Run (Frontend Only)
Before starting, ensure the backend is running.
1. Install dependencies:
```
   npm install
```

2. Start the development server:
 ```
    npm start
```

### Tech Stack
- Framework: React 19
- Styling: Tailwind CSS (via CDN in index.html)
- Icons: Lucide-React
- Charts: Recharts
